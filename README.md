# Python Data Cleaning Project

This project demonstrates cleaning a real-world messy CSV file using Python and pandas.

## Problems handled
- Removed duplicate rows
- Cleaned and standardized column names
- Fixed currency values in numeric fields
- Handled missing data
- Standardized text values

## Project files
- messy_data.csv – raw input data
- clean_data.py – Python script for data cleaning
- cleaned_data.csv – cleaned output file

## Tools used
- Python
- Pandas

## Outcome
The cleaned dataset is ready for reporting or analysis.
